## Grizzly + Jersey

```bash 
cd grizzly-jersey/
mvn clean install
cd target/
unzip grizzly-jersey-proto-0.0.1-SNAPSHOT-distribution.zip
cd grizzly-jersey-proto-0.0.1-SNAPSHOT/
java -jar grizzly-jersey-proto-0.0.1-SNAPSHOT.jar
```

Zip file size 5.3 MB.
